# T-Rex

A script for inserting a new page in a number-indexed Docusaurus directory as shown [here](https://docusaurus.io/docs/sidebar/autogenerated#using-number-prefixes).

---

## Installation (soontm)

### From crates.io:
```bash
cargo install t-rex
t-rex --help
```
